## How to upload your PPTX and MP4

Drop the two files into this folder using **exactly** the filenames below so
the links in the main `README.md` resolve without any further edits:

```
/app/documentation/TriageAI_Presentation.pptx
/app/documentation/TriageAI_Demo.mp4
```

If you prefer different filenames, just tell us and we will update the links
in [`/app/README.md`](../README.md) accordingly.

> **Tip:** GitHub renders MP4 files inline in the README when the file lives
> in the same repository, and PPTX files become a one-click download link.
> No external CDN is required.
